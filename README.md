# Sports Hall Booking System

## Overview

The **Sports Hall Booking System** is a ServiceNow-based application developed to simplify the process of reserving sports facilities. It provides an easy-to-use Service Portal where users can view available sports halls and submit booking requests, while administrators can efficiently manage bookings and prevent scheduling conflicts.

## Features

* Sports hall booking through Service Portal
* User-friendly booking interface
* Duplicate booking validation
* Custom ServiceNow widget
* Business Rules for booking management
* Event Registry integration
* Scheduled Jobs for automated tasks
* Email Notifications for booking events
* Responsive user interface

## Technologies Used

* ServiceNow Platform

## Project Workflow

1. User accesses the Service Portal.
2. User selects a sports hall and enters booking details.
3. The system validates the booking request.
4. Business Rules prevent duplicate bookings.
5. Booking information is stored in ServiceNow.
6. Notifications and scheduled processes handle booking-related events.
